# Simple Timer App
This is a static timer app with a count down or count up timer app.

## Features
1. It shows Hours, Minutes and Seconds
2. It can count down as well as count up
3. It saves the current time and settings in the local storage so that refresh doesnt affect the timer to restart
4. Options to set alarms at specific duration like 1:00:00, 2:15:00 etc.
5. The hours is only shown when the timer has hours, similarly for minutes.
6. If the count down or count up expires an alarm must ring.
